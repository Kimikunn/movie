# 功能文档

> 基准：`app/src` 当前实现（更新于 2026-03-04）

## 路由总览

- `/home`：首页（HomePage）
- `/search`：搜索影片页（ProfilePage）
- `/records`：观影记录统计页（DashboardPage）
- `/records/month`：月详情页（RecordsMonthPage）
- `/records/week`：周详情页（RecordsWeekPage）
- `/detail`：影片详情页（MoviesPage）
- `/create`：兼容路由，重定向到 `/search`
- `/`：重定向到 `/home`

兼容别名：

- `/movies` -> `/home`
- `/profile` -> `/search`
- `/dashboard` -> `/records`

## 数据来源

- 本地静态数据：`HomePage`、`DashboardPage`、`movieLibrary`
- URL 查询参数：`from`、`movie`、`q`、`mode`
- 本地存储：`localStorage(movie.watchLogs.v1)`（新增记录）
- 本地存储：`localStorage(movie.shortReviews.v1)`（短评）

## Page 0：全局壳层（AppShell）

- 状态栏时间、系统图标、内容容器、底部悬浮 Tab（仅 icon）
- Tab 路由：`/home`、`/search`、`/records`

## Page 5：周详情页（`/records/week`）

- 顶部导航（返回 `/records/month` + 分享）

## Page 6：影片详情页（`/detail`）

- 顶部导航（返回 + 分享）
  - `from=search` -> 回 `/search`（保留 `q`）
  - `from=records-month` -> 回 `/records/month`
  - `from=records-week` -> 回 `/records/week`
  - `from=records` -> 回 `/records`
  - 其他 -> 回 `/home`
- 影片 Hero 信息卡
- 我的评分对比卡
- 平台评分区
- 影片简介区
- 观影记录区（读取默认记录 + 本地新增记录）
- 我的短评区

### 4.1 新增记录 Bottom Sheet

- 触发入口：详情页「观影记录」模块右上角“添加”按钮
- 展现形式：从底部弹出，约半屏高度（可上拉），不是页面跳转
- 视觉规范：iOS26 风格半透明毛玻璃（blur + 轻边框 + 阴影）
- 字段：观影日期、观影地点与方式（同一输入块，分段选择含“流媒体”选项）、我的评分（进度条+数字输入联动）
- 提交行为：
  - 校验必填（日期/地点/评分）
  - 保存到 `localStorage(movie.watchLogs.v1)`
  - 关闭弹窗并即时刷新当前详情页观影记录列表

### 4.2 我的短评编辑（详情页内）

- 触发入口：详情页「我的短评」卡片，点击卡片直接进入编辑
- 展现形式：在当前短评卡片内进入编辑态（textarea + 取消/保存）
- 保存位置：`localStorage(movie.shortReviews.v1)`（按 movieId 存储）

### 4.3 观影记录修改（详情页内）

- 触发入口：详情页「观影记录」中每条本地记录，点击记录项直接进入修改
- 展现形式：复用同一个 Bottom Sheet，自动回填该条记录
- 提交行为：保存后覆盖本地该条记录，并即时刷新列表
- 数据范围：仅本地新增记录可修改（默认静态记录不显示修改入口）
